# assets/css

## Função

Estilos do site, organizados por responsabilidade.

## Estrutura

- `main.css` — ponto de entrada, importa os demais.
- `base/` — reset, variáveis, tipografia.
- `layout/` — header, footer, seções.
- `components/` — botões, cards, navegação.
- `pages/` — estilos específicos de página (`home.css`).

## Convenção de nomes

kebab-case, em inglês, descritivo (ex: `buttons.css`).
