# Site Lideranças NMI

Site estático do curso de Lideranças do NMI (Núcleo de Estudos em Políticas, Direito, Economia e Tecnologia das Comunicações) , construído com HTML, CSS e JavaScript puro para publicação no GitHub Pages.

Layout implementado a partir do design Figma: `Site-Lideranças-NMI` (node `1:2685`).

## Estrutura

```text
index.html
assets/
  css/        estilos (base, layout, components, pages)
  js/         scripts (app.js + components/pages)
  img/        logos, favicon, backgrounds
docs/         documentação técnica do projeto
```

## Como rodar localmente

Basta abrir `index.html` no navegador, ou servir a raiz com qualquer servidor estático (ex: `npx serve .`).

## Publicação

GitHub Pages configurado para publicar a partir da raiz do repositório (`index.html`).
