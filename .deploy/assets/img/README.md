# assets/img

## Função

Imagens públicas usadas pela interface.

## Estrutura

- `logo/` — logo principal, marca do footer, selos e logos de apoio.
- `favicon/` — favicon do site.
- `backgrounds/` — imagens decorativas de fundo.
- `icons/`, `illustrations/` — reservado para crescimento futuro.

## Observações

Ativos originados do Figma (`Site-Lideranças-NMI`, node `1:2685`) via MCP; URLs de exportação do Figma expiram em 7 dias, por isso os arquivos foram baixados para esta pasta.
