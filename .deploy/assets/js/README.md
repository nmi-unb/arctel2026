# assets/js

## Função

Comportamento e interatividade do site, via ES Modules puros (sem bundler).

## Estrutura

- `app.js` — ponto de entrada, carregado com `type="module"`.
- `components/` — comportamento reutilizável (ex: acordeão de módulos).
- `pages/` — inicialização específica de cada página.

## Convenção de nomes

kebab-case, em inglês (ex: `module-accordion.js`, `home-page.js`).
